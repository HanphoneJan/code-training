# 字符串

## 知识点概述

字符串是字符序列的抽象表示，常见操作包括索引、匹配、分割和拼接。

## 常见考点

- 双指针与滑动窗口
- 子串与子序列
- 字符计数与映射

[几道常见的字符串算法题 | JavaGuide](https://javaguide.cn/cs-basics/algorithms/string-algorithm-problems.html)

- **BF（暴力匹配）**：O (nm)，简单但效率低。
- **KMP 算法**：字符串模式匹配，利用前缀函数避免重复匹配，O (n+m)。
