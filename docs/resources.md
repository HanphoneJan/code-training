## 学习建议

1. **循序渐进**：按照灵神路线图，从基础数据结构开始，逐步过渡到复杂算法
2. **五遍刷题法**：
   - 第一遍：理解题目，能写出解法即可
   - 第二遍：优化代码，争取最优解
   - 第三遍：一天后复习
   - 第四遍：一周后复习
   - 第五遍：面试前复习
3. **可视化学习**：利用 VisuAlgo 和 Python Tutor 理解算法执行过程
4. **归纳总结**：每类题型整理出通用的解题模板和套路
