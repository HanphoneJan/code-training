# 欢迎来到 Code Training

这是一个**系统化的算法训练知识库**，采用数据库思维构建，面向长期学习和持续积累。

## 📚 核心模块

### [题目库](./problems)
每道题目独立存档，包含完整的题目描述、解题思路、代码实现和相关链接。

- **LeetCode** - 主要刷题平台
- **牛客** - 面试题库
- **其他** - 各类算法题目

### [知识点](./topics)
按数据结构和算法分类的知识体系。

- 数组、链表、栈、队列
- 树、图、哈希表
- 排序、查找、动态规划

### [算法模式](./patterns)
通用的解题模式和思维框架。

- 双指针、滑动窗口
- DFS、BFS
- 回溯、贪心

### [代码模板](./templates)
常用算法的标准实现，开箱即用。

- BFS/DFS 模板
- 二分查找模板
- 并查集模板

### [复习系统](./review)
错题本和周总结，强化记忆。

- 错题记录与分析
- 每周学习总结
- 复习计划追踪

## 🚀 快速开始

1. **添加新题目** - 在 `problems/平台/` 下创建题目文件
2. **学习知识点** - 浏览 `topics/` 目录系统学习
3. **掌握模式** - 查看 `patterns/` 学习解题套路
4. **使用模板** - 从 `templates/` 复制常用代码
5. **记录复习** - 在 `review/` 中记录错题和总结

## 📖 使用指南

### 题目文件格式

每个题目文件包含 YAML frontmatter 和 Markdown 内容：

```yaml
---
title: 题目名称
platform: LeetCode
difficulty: 简单/中等/困难
tags: [数组, 哈希表]
topics:
  - ../../topics/array.md
patterns:
  - ../../patterns/hash_map.md
---
```

### 知识网络

通过相对路径链接形成知识网络：
- 题目 → 知识点
- 题目 → 算法模式
- 知识点 → 相关题目
- 模式 → 练习题目

## 🎯 学习路径

### 入门阶段
1. 数组 + 哈希表
2. 字符串 + 双指针
3. 链表基础

### 进阶阶段
4. 树的遍历（DFS/BFS）
5. 二叉搜索树
6. 图算法

### 高级阶段
7. 动态规划
8. 回溯算法
9. 高级数据结构

## 💡 设计理念

> **数据库思维，不是平台思维**

- ✅ 原子化存储：每题独立
- ✅ 知识聚合：按知识点组织
- ✅ 双向链接：形成知识网络
- ✅ 持续积累：长期维护

## 📊 进度统计

使用左侧导航栏快速访问各个模块，或使用顶部搜索（`Ctrl+K` / `Cmd+K`）查找内容。
